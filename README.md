# React-Wood
> A tree view component for React.
[![NPM](https://img.shields.io/npm/v/react-wood.svg)](https://www.npmjs.com/package/react-wood)

## Installation
### NPM
```sh
npm install --save react-wood
```

**or**

### Yarn
```sh
yarn add react-wood
```
